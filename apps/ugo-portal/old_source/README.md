# old_source

このディレクトリには、ディレクトリ構造ルール（docs/directory-structure.md）に準拠していない既存のコードを一時的に格納しています。

## 移動したディレクトリ

### components/
- **元の場所**: `src/components/`
- **移動理由**: ルール違反（アプリ内のコンポーネントは`pages/{ページ名}/_components/`または`src/feature/`に配置すべき）
- **今後の対応**: 必要に応じて適切なディレクトリに再配置

### contexts/
- **元の場所**: `src/contexts/`
- **移動理由**: ルール違反（contextは`src/feature/`に配置すべき）
- **今後の対応**: 必要に応じて`src/feature/`に再配置

## 注意事項

- このディレクトリのコードは現在使用されていません
- 新しいコードはこのディレクトリを参照しないでください
- 将来的なリファクタリング時の参考資料として保管しています
